## $${\color{pink} Optimising \space Non-Pharmaceutical \space Intervention \space Policies \space for \space Respiratory \space Virus \space Outbreaks}$$ <br />  $${\color{pink} in \space the \space Presence \space of \space Future \space Vaccine \space Uncertainty}$$

Python code for various comparmental models (SEIHR and SEIHRV) for a respiratory virus outbreak in the UK. Vaccine uncertainty and deployment day can be varied to observe the effect on the outbreak.
Non-pharmaceutical interventions (NPIs) are then considered in the form of a lockdown, where the start, duration, and intensity of the lockdown can be varied.
